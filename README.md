# Alphonso---A-Complete-MangoWC-setup-files

Based on actual mango varieties found around the world
Includes-
Wallpapers
Waybar
Lockscreens (via Hyprlock)
a mangoWC config file that is easy to setup and use
customised terminal apps such as btop, etc

Theme switcher-
Alphonso (Default)
Blush Alphonso
Kesar
Marshland
Palmer

There is an install script as well, but it is not yet concrete, so its best to install it manually.
